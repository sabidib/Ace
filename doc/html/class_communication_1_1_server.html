Detailed Description

Represents the implementation of an internet server. This will bind the given socket to the local machine. The server can be set to listening mode by using the Server::startServer() and can wait for incoming connections with Server::startAccepting(). Finally, the server can be shutdown by using Server::shutdownServer();.

Ownership of the listening socket can be modified according to the Server::relinquishOwnership() and Server::transferOwnership() functions.
